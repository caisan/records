## 1. POM Reference

### 1.1 POM介绍

###### 1.1.1 什么是POM?

POM是工程对象模型(Project Object Model)的缩写。其是一个Maven工程的XML表示形式，该XML的名称为pom.xml。通常一个project包含配置文件、所涉及到的开发者以及他们所扮演的角色，
